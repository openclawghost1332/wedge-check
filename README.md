# Wedge Check

Wedge Check is a tiny static web app that pressure-tests a product idea into a sharper wedge.

## What it does

- scores customer specificity, current workaround, demand evidence, and wedge sharpness
- surfaces red flags and repair advice when an idea is too fuzzy
- rewrites rough inputs into a cleaner one-line wedge statement
- runs entirely client-side, no API key required

## Local development

```bash
npm test
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.
